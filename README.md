# Pathfinding Visualizer

## Installation

```bash
git clone https://github.com/andrewywong/pathfinding-visualizer
cd Pathfinding-Visualizer
npm install
npm start
```

## Algorithm List

- Dijkstra's algorithm

- Breadth-first search

- Depth-first search

- A\* search algorithm
